# Minhas Notas De Programação 

Este repositório foi criado no para reunir minhas anotações de estudo em **programação**. 

Organizo tudo no Obsidian e sincronizo aqui para ter um histórico da minha evolução e facilitar revisões futuras. 

----

## Objetivo do Repositório 

 - Centralizar todo o meu conteúdo de estudo
 - Registrar minha evolução como desenvolvedor
 - Criar uma base de conhecimento 
 - Compartilhar minhas anotações com recrutadores e outros devs
 - Manter uma organização para consultas rápidas

---

## Como organizo minhas anotações 

Uso um sistema dividido em **Rough Notes** e **Full Notes** para garantir clareza, revisão eficiente e conseguir aprender de verdade. 

### Rough Notes (Notas Rápidas)

Anotações feitas **durante** o estudo, enquanto assisto uma aula ou leio um conteúdo.

- Tópicos rápidos 
- Palavras-chave
- Exemplos curtos
- Rascunhos sem formatação 

**Objetivo**: Capturar tudo rápido, sem travar tentando deixar bonito e estruturado. 

----

### Full Notes (Notas Completas)

Criadas **depois** do estudo, com calma.
É onde realmente consolido o conhecimento e deixo mais estruturado. 

- Explicações completas com minhas palavras
- Estrutura organizada em tópicos 
- Exemplos explicados 
- Links de referência 
- Resumo final

**Objetivo**: Transformar o conteúdo em algo claro e que eu possa entender daqui a 1 mês ou 1 ano.
