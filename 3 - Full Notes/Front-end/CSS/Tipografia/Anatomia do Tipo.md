# Conceito Geral

Já sabemos que tipografia é o estudo dos tipos de letras, mas quando vão criar uma fonte são feitas várias medidas, não só o tamanho da fonte. 

A primeira coisa que fazem quando vão criar uma fonte nova, e que é muito importante é a medida do **x** minúsculo, pois existe uma métrica chamada **Altura X**, a altura do x minúsculo vai definir qual é a base de todas as letras minúsculas dessa fonte. 

![[Pasted image 20251126142743.png]]

- **Altura x**: Altura base para as letras minúsculas. 
- **Altura das maiúsculas**: Altura das letras maiúsculas. 
- **Ascendente**: É a parte da letra minúscula que vaza pra cima, acima da altura X, como o b, k e d.
- **Descendente**: É a parte da letra minúscula que vaza pra baixo, abaixo da altura X, como g, p, q.
- **Corpo**: É a soma de tudo. 

#### Serifas 

![[Pasted image 20251126143546.png]]

Serifa é esse "pezinho" que fica na letra, ou em baixo ou em cima, e serve para o nosso cérebro criar uma linha, e facilitar a leitura.

## Famílias tipográfica

![[Pasted image 20251126145724.png]]

É a mesma fonte, só que em famílias tipográficas diferentes. 

Nem toda fonte tem família tipográfica, todas tem o **normal**, e normalmente o **negrito**, também mas existem fontes que não tem o negrito. 

## Categorias de Fontes

![[Pasted image 20251126151942.png]]

- **Monoespaçadas**: Aquelas que cada glifo ocupa exatamente o mesmo espaço, tem a mesma largura;
- **Handwriting**: São letras que simulam glifos feitos a mão 
- **Display**: São aquelas que não tem nada das de cima, não se preocupa em obedecer as regras anatômicas.
